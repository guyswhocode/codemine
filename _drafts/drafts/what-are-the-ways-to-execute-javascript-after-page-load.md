---
title: What are the ways to execute javascript after page load 
categories: ["programming"] 
tags: ["JS"]
---
     You can execute javascript after page load in many different ways,
     1. ** window.onload:**
     ```javascript
     window.onload = function ...
     ```
     2. **document.onload:**
     ```javascript
     document.onload = function ...
     ```
     3. ** body onload:**
     ```javascript
     <body onload="script();">
     ```
